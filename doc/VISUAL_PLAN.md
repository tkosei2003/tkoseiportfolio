# ヴィジュアルアイデア(PC版)
＊モバイル版のために拡張性を持たせる（各セクション/コンポーネントのモジュール化）
## 0. 全体像
| セクション | UX意図 | 技術メモ |
| --- | --- | --- |
| Hero | ポインタ反応するロゴ/粒子で第一印象を作る | ビューポート全体Canvas、カスタムシェーダ or InstancedMesh |
| About | 自己紹介＋数値ハイライト | 標準React構成、軽めのモーション |
| Timeline/Skills | スクロールで順次表示、スキル強調 | Framer Motionのスタッガー、3Dオービット演出も検討 |
| Projects | 3D/動画と連動したカード表示 | MDX駆動、必要に応じGLTFプレビュー |
| Contact | フォーム＋SNS、3Dキャラ演出 | Server Action送信、公開時はreCAPTCHA等 |
## 1. Hero
- 名前の英語表記（Takaishi Kosei）を3D文字で大きく中央に表示
  - 名前の3D文字は粒子でできておりポインタに反応
    - ポインタがある場所は疎になる
- 左下にスクロールの指示（下向き矢印の２Dアニメーション）
- GPU負荷が高いので、Points＋InstancedBufferGeometry＋カスタムshaderMaterialで1ドローコール化し、prefers-reduced-motion検出時は静的テキストにフォールバックする
## 2. About
- 自己紹介を名刺ふうに
- 名刺カードは3Dインタラクティブ
- カードはコンポーネント化して、他セクションでも使えるようにする。（以下、浮遊カード）
    - カード構造・データスキーマ・モーション設定をコンポーネント化し、MDX/JSONでデータ駆動にする
- 中央に名刺が浮かんだ形、カーソルに反応して少し傾く。
- Aboutのセクション見出しは縦に入れるのがテーマに沿いそう？（要検討（トライアンドエラー））
- シンプルな自己紹介文
    - 内容：兵庫県出身のエンジニアの卵。大阪公立大学にて情報工学を勉強中。研究では主にpython、
    アルバイトではpython、flutter、
    個人開発＆趣味でnext.js、three.js、blender、3Dプリンター
    を活用している。
    - 名刺のようなカードの上に印刷された形で、カード自体は3Dオブジェクトとして角度がカーソルに合わせて見えにくくならない程度に傾いたりする。
    - 文字もカードと連携して動く。
- 背景に立体物を浮かばせてインタラクティブなスクロールを実現
    - 自己紹介文に干渉しない程度のシンプルなもの
    - Canvasを1つに集約しScrollControlsでシーンを分割する
## 3. Timeline/Skills
### Timeline
- Timelineは蔦/ツリーが浮いているようなイメージで、各枝葉に内容の省略表示が浮かぶ。
- 更新を見据えて適宜コンポーネント化。
- 各葉にはカーソルホバーで何かクリックできることを予測させるようなアクションをつける
- クリックで葉にズームインして詳細内容が表示
### Skills
- 画面中央に各技術を示す立体アイコンが輪の形で宙に浮いている
    - pokepokeのカードパック選択画面みたいな
- ドラッグで輪を横スクロールできる
- アイコンをクリックすると
    - アイコンの輪が縮小して画面右に移動
    - 左側に、その技術をどれぐらい使っているか、詳細が表示（浮遊カード）
        - 技術名
        - 使用年数
        - その技術を使っている、後のプロジェクトへのリンク
### アクセシビリティ
- キーボード操作とスクリーンリーダー向けに aria-roledescription や focus 時のズーム遷移を用意する
## 4. Projects
- 同様にポケカ方式で浮遊カードを輪に並べて配置
- 各カードに
    - プロジェクト名
    - プロジェクト内容の略記
    - 象徴的な３Dアイコン（オプション、カードから飛び出してくる感じ）
- 複数同時表示されるため、LODやドラコ圧縮済み低ポリモデルを使い、CPU/VRAM目安（例: 2Mポリゴン/50MB以下）
## 5. Contact
- シンプルなアイコンとメールアドレスのみやすい並び
- 他セクションから独立した背景色、２D表示
- ２Dののっぺりとした背景にぶら下がる形で３Dキャラがこちらを覗く
- 別Canvasを重ねるのではなくSVG＋CSS 3Dに置き換える　or　WebGL側をOffscreenCanvasで遅延読み込みする
